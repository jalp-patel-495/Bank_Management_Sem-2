# 🏦 Bank Management System

Bank Management System is a Java console-based application that simulates basic banking operations using Object-Oriented Programming (OOP) concepts.  
This project is created as a semester project to demonstrate OOP principles and fundamental banking workflows.

----

## 📌 Project Overview

This Bank Management System allows the user to:

✔ Create a bank account  
✔ Deposit money  
✔ Withdraw money  
✔ Handle loans  
✔ Manage fixed deposits  
✔ Different user roles such as User, Staff & Bank Manager

The system is built in **Java** and focuses on clean, modular, readable code based on OOP principles. :contentReference[oaicite:1]{index=1}

## 🧠 OOP Concepts Used

- **Inheritance** – for extending base account functionality  
- **Encapsulation** – using private variables with getters/setters  
- **Polymorphism** – method overriding

## ⚙️ How to Run the Project

### 📌 Prerequisites

✔ Java JDK 8 or above  
✔ Any Java IDE (VS Code, IntelliJ, Eclipse) :contentReference[oaicite:5]{index=5}

### 🏃 Compile & Run

1. Clone the repository:
   ```bash
   git clone https://github.com/jalp-patel-495/Bank_Management_Sem-2.git
   ```

2. Go inside the project folder:
   ```bash
   cd Bank_Management_Sem-2
   ```

3. Compile the Java files:
   ```bash
   javac Main.java
   ```

4. Run the project:
   ```bash
   java Main
   ``` :contentReference[oaicite:6]{index=6}
